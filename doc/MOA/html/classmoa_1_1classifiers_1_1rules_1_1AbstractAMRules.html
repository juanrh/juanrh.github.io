AMRules Algorithm

For each rule in the rule set If rule covers the instance if the instance is not an anomaly Update Change Detection Tests Compute prediction error Call PHTest If change is detected then Remove rule Else Update sufficient statistics of rule If number of examples in rule > Nmin Expand rule If ordered set then break If none of the rule covers the instance Update sufficient statistics of default rule If number of examples in default rule is multiple of Nmin Expand default rule and add it to the set of rules Reset the default rule
