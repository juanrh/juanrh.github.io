Detailed Description

Class for using a MEKA classifier.

NOTE: This class only exists to adjust the classIndex by +1 We can use the standard WEKAClassifier if we set -c L where, L = the number of labels + 1 (Because MOA understands that L specified on the command line is the (L-1)th index).
