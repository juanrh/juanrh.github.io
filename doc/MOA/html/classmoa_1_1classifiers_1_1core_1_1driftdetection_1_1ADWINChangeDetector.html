Detailed Description

Drift detection method based in ADWIN.

ADaptive sliding WINdow is a change detector and estimator. It keeps a variable-length window of recently seen items, with the property that the window has the maximal length statistically consistent with the hypothesis "there has been no change in the average value inside the window".
