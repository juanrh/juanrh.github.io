Detailed Description

A set of DataObjects.

Internally it uses an ArrayList<DataObject> so it is recommended to use with care. For speedy operations after the data is collected use getFeaturesAsArray(). When access speed is not a factor the iterator can be used to go through the data with a foreach loop.
